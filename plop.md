# Use plop to create a new file

```bash
npm install --save-dev plop
```

# create a screen

```bash
npx plop screen
```

# create a component

```bash
npx plop component
```

# create a redux

```bash
npx plop redux
```

# trước khi tạo bottom tab, cần tạo các file sau trước

```bash
npx plop config
npx plop ui-global
npx plop nav-services
```
